修改配置文件添加用户

发表于 2016-07-06   |   分类于 linux   |  

用户配置文件

输出7个字段，用冒号分割。
| 字段 | 一个普通标题 |
| ——| —— |
| root: |用户名 |
| x:|密码占位符，加密后的密码存放在/etc/shadow |
|0:|user id 普通用户>500 ,系统用户<500| |0:|group="" id="" 普通用户="">500 ,系统用户<500|
|root:|用户描述信息|
|/bin/bash:|用户登陆系统时使用的shell|

/etc/shadow 存放与用户密码相关的参数
|root:|用户名，与/etc/passwd中对应|
|字段2：|加密算法加密的用户密钥，如果为！！ 表示没有创建用户密码|
|16912：|最后修改密码日期距离1970年1月1日的天数。|
|0：|指定密码修改多少天后可以修改|
|99999：|多少天后必须修改密码|
|7：|密码到期后7天还没修改，账户失效|
|：|账户失效时间距离1970年1月1日的天数，一般用于收费系统|
|：|保留字段|

|root:|组名称|
|x：|组密码|
|1：|组id|
|:|属于改组的用户，使用都好分割|

添加用户修改/etc/passwd/

51cto:x:1003:1003::/home/51cto:/bin/bash

添加用户修改/etc/group文件

51cto:x:1003:51cto

添加用户修改/etc/shadow文件

/etc/shadow
文件末尾添加: 51cto:$1$123456$wOSEtcyiP2N/IfIl15W6Z0:16988:0:99999:7:::
